// Find first name of employees with age less than 30 and salary greater than 50000
db.Employee.find(
{
Age:{$lt:30},
Salary:{$gt:50000}
},
{
"Name.FName":1,
_id:0
}
)




// Create new document if matching document is not present
db.Employee.updateOne(
{
Designation:"Tester",
Company_name:"TCS",
Age:25
},
{
$set:{
Name:{FName:"Kiran",LName:"Patil"},
Salary:32000
}
},
{
upsert:true
}
)




// Find employees where age is less than 30 or salary greater than 40000
db.Employee.find(
{
$or:[
{Age:{$lt:30}},
{Salary:{$gt:40000}}
]
}
)




// Display employees whose designation is not Developer
db.Employee.find(
{
Designation:{$ne:"Developer"}
}
)

Aim

To study MongoDB CRUD operations using Employee collection and query operators.

Objective
To perform CRUD operations in MongoDB.
To understand projection and conditional operators.
To work with embedded documents and arrays.
To retrieve selected fields from documents.
Theory

MongoDB is a NoSQL database that stores data in the form of documents and collections.

CRUD operations in MongoDB are:

Create
Read
Update
Delete

Functions Used:

insertMany() → Inserts multiple documents
find() → Retrieves documents
updateOne() → Updates one document

Operators Used:

$lt → Less than
$gt → Greater than
$or → Logical OR
$ne → Not equal
upsert:true → Insert if document not found

Projection:
Projection is used to display selected fields from documents.

Advantages of MongoDB:

Flexible schema
High performance
Easy to store nested documents
Suitable for large scale applications
Conclusion

Thus, MongoDB CRUD operations were successfully performed using conditional operators, projection and embedded documents.
